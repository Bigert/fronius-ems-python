# fronius-ems-python

A Python client library for the Fronius GEN24 Energy Management System (EMS) Web API.

## Goals

- Authenticate against the GEN24 Web API.
- Read EMS configuration and status.
- Update Soft Export Limit.
- Provide a stable API for Home Assistant and other automation platforms.

## Project status

This project is under active development.

## Planned features

- SHA-256 Digest authentication
- Power Limits API
- Device Status API
- Power Unit API
- Comprehensive logging and diagnostics
- Unit tests
